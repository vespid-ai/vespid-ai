# Why this stack fits vespid.ai

A personal engineering site usually dies in one of two ways:

- it becomes visually polished but hard to update
- it becomes easy to update but visually forgettable

Astro + Starlight is the compromise that still feels ambitious.

## Astro handles

- homepage composition
- design freedom
- future custom sections
- polished presentation

## Starlight handles

- navigation
- content organization
- search
- code-friendly docs
- the boring parts that should stay boring

That split is exactly what this site needs.
