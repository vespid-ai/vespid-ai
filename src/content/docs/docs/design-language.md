# Design Language

The chosen direction is a hybrid, not a clone.

## Structural shell — Linear

Linear remains the best reference for an operator-grade dark interface:

- precise spacing
- calm hierarchy
- sharp cards
- limited accent palette

## Showcase energy — Framer

Framer is the reference for the hero experience:

- larger display type
- product-first storytelling
- stronger sense of motion and confidence

## Reading ergonomics — Mintlify

Mintlify is the reference for the documentation layer:

- easier scanability
- clean card groupings
- calmer long-form reading rhythm

## Practical implication

The vespid.ai site should feel premium and current, but the copy must still carry the page. Effects support the content; they do not replace it.
